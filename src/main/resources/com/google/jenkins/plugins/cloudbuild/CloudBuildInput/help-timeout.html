The maximum time the build can last before it should fail as TIMEOUT. If specified, this overrides the timeout specified in the build request (if any). This field is case-insensitive, and must be expressed in one of the following forms:
  • An ISO-8601 Duration not containing elements larger than days (e.g., P1DT3H14M15.9S). Days are interpreted as exactly 24 hours.
  • A string of the form nHnMn.nnnS indicating hours, minutes, and seconds (e.g., 3h14m15.9s). Zero-valued segments may be omitted (e.g., 14m or 3h15.9s). Note that this is equivalent to an ISO-8601 duration with the prefix "PT" omitted.
  • A number (with an optional decimal part). This is interpreted as the specified number of seconds. Note that this is equivalent to the ISO-8601 duration "PTn.nnnS".
